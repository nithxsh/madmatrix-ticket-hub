
# MadMatrix '26 Ticket Hub

Authorized ticket retrieval portal for MadMatrix 2026. This application is built with Next.js, Tailwind CSS, and Genkit AI.

## Deployment & Version Control

To push this project to your GitHub repository, execute the following commands in your terminal:

### 1. Initialize Git
```bash
git init
```

### 2. Stage and Commit
```bash
git add .
git commit -m "Initial commit: MadMatrix 2026 Portal"
```

### 3. Connect to GitHub
```bash
git branch -M main
git remote add origin https://github.com/nithxsh/madmatrix-ticket-hub.git
```

### 4. Push to Cloud
```bash
git push -u origin main
```

## Features
- **Neural Launch Sequence**: Tactical "Slide to Launch" interaction.
- **Vertical Data Stream**: High-contrast terminal-style identity verification.
- **High-Res PDF Permits**: Absolute coordinate hard-locked ticket generation at 3x resolution.
- **Responsive Design**: Fully optimized for mobile and desktop viewing.

## Development
To start the development server:
```bash
npm run dev
```
